# EKA - Executable Knowledge Architecture

## About EKA and its Name (关于EKA的缩写)

使用EKA缩写的词汇主要有如下几种，见表格：

| 全称 | 释义与考量 |
| --- | --- | 
| Enterprise Knowledge Architevcture (企业知识架构) | 这应该是最接近和最容易混淆使用的缩写，这个词汇在 ATHENA 互操作性项目等学术研究中，被定义为“一种支持企业间协作建模、元建模和模型执行的框架”。区别在于这个E现在我用来突出强调 Executable （可执行）的意味，而弱化了“企业”这个词汇的单独出现，其实企业也好、组织也好，都是可以隐含在架构这个词后面的。 |
| Embedded Karmarkar Algorithm (嵌入式卡玛卡算法) | 由印度 Computational Research Laboratories 建造的超级计算机，曾在2007年被评为全球第四快、亚洲第一快。 这个算法是这台超级计算机名字的来由，指一种用于线性规划的高效算法。 |
| Mendeleev 的前缀 (Eka-) | 在化学中，"Eka-"是一个著名的梵语前缀 |
