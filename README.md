# EKA - Executable Knowledge Architecture

## About EKA and its Name (关于EKA的缩写名字的释义)

使用EKA缩写的词汇主要有如下几种，见表格：

| 全称 | 释义与考量 |
| --- | --- | 
| Enterprise Knowledge Architevcture (企业知识架构) | 这应该是最接近和最容易混淆使用的缩写，这个词汇在 ATHENA 互操作性项目等学术研究中，被定义为“一种支持企业间协作建模、元建模和模型执行的框架”。区别在于这个E现在我用来突出强调 Executable （可执行）的意味，而弱化了“企业”这个词汇的单独出现，其实企业也好、组织也好，都是可以隐含在架构这个词后面的。 |
| Embedded Karmarkar Algorithm (嵌入式卡玛卡算法) | 由印度 Computational Research Laboratories 建造的超级计算机，曾在2007年被评为全球第四快、亚洲第一快。 这个算法是这台超级计算机名字的来由，指一种用于线性规划的高效算法。 |
| Mendeleev 的前缀 (Eka-) | 在化学中，"Eka-"是一个著名的梵语前缀， 由门捷列夫提出，用于命名元素周期表中尚未发现的元素。例如，**Eka-silicon**指的是后来发现的锗 (Germanium)。在科学语境下，EKA往往暗示“某种事物的先行版或预测版”。 |
| Eesti Kunstiakadeemia (爱沙尼亚艺术学院) | 该学院的官方缩写即为 EKA |
| Eka language | 一种分布在中国境内的哈尼语支语言的名称 |
| EKA Architects | 一家知名的建筑事务所 |

从语言的角度，很凑巧和梵语中的Eka有所关联，在梵语中，`Eka`的写法如下：
- 天城文（Devanagari）： एक
- 拉丁字母转写（IAST）：eka

在梵语中，这个词有很深厚的哲学与数学背景，这方面与我的EKA所致力于的`架构统一性`非常契合：
- 数字含义：它代表数字"`1`"。
- 哲学含义：它象征着“唯一”、“统一”或“单一”。在许多古代文献中，它被用来描述宇宙的本意或万物的统一性。

从这个意义上来讲，梵语中的“统一”含义与我的EKA的全程刚好完美契合：

> EKA不仅代表“可执行”，更代表了将**静态文档、动态数据与逻辑执行**高度**统一(Unity)**的架构哲学。
